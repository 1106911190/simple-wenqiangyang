### Python 的类的常见内置方法和属性

#### 类的属性

- __dict__  : 类的属性（包含一个字典，由类的数据属性组成）
- __doc__ : 类的说明文档，即定义类时的注释
- __name__ : 类的名字
- __file__ : 类所在的完整文件地址
- __bases__ : 类的父类
- __module__ : 类所在的模块
